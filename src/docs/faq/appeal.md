# 幻梦拉黑常见问题

### 我怎么被幻梦屏蔽了？

![#*px #350px](/img/appeal_main.png)

小人类您好，感谢查看本篇“常见问题”，请您检查最近是否有以下行为：
1. 辱骂幻梦。如您在群内发表（包括曾经发表）辱骂幻梦的言论，或者您在调用指令时，指令文本包含辱骂的词汇等。
2. 利用作弊、漏洞等来获得不正当的游戏优势，如:
- 使用按键精灵、映射、辅助点击等自动软件或脚本游玩幻梦小游戏或者其他功能。
- 利用某小游戏玩法的漏洞来恶意刷取大量额度、好感度、梦晶、道具、称号的行为。
- 等等......
3. 调用指令频率过高或有异常。短时间内非常频繁地调用幻梦的指令，或者检测到您在过去24小时内，每X分钟就有规律地调用幻梦一次指令等异常行为。
4. 大量无意义的指令调用行为。如多次尝试各个本身不存在的指令（如 /plugins 等）以了解幻梦可能用了什么插件，可能用的是什么模块或者框架等。
5. 消极行为。如发送“不玩了”、“别发了”、“停”等；或发送“退坑了”、“踢了”、“骂她”等消极话语。
6. 违规发表言论。如在今日超能力、剧情模式的评论区发表涉政、涉黄等违规评论，或有其他尝试让幻梦说出违规言论的行为。
7. 违规自主输入。如在人设生成玩法中设定违规人设（名字、爱好、血型等自主输入涉及敏感词等），上传违规头像等。
8. 其他不良行为。
- 在社交平台、视频平台上发布关于幻梦的不实言论（简称造谣）。
- 在开发您自己的项目时，存在恶意抄袭/搬运/模仿幻梦的任一已上线功能。
- 通过不正当手段获取幻梦运行所需要的源码。
- 通过批量调用幻梦的查询指令功能来收集大量数据。
- 其他违反了《用户守则》的内容等等。

### 如何解除幻梦对我的屏蔽？
- 无论做错了什么，只要承认错误并道歉就有机会解除屏蔽，从而继续游玩幻梦。
- 现在请小人类任意调用幻梦的一个指令，幻梦将会回复您关于被拉黑的消息。（如果幻梦因上述3、4点不回复您时，请直接看下面）
    
![](/img/appeal_list_commands.png) ![](/img/appeal_send_any_message.png) ![](/img/appeal_join_official_group.png)

- 查看下方的按钮，如果可以立刻申请解封，则可按照指引一步一步申请解封。
    - 如果不能申请立即解封，就请点击下面的按钮前往幻梦官方群向幻梦开发组说明最近使用幻梦情况。

如果小人类觉得自己是被误封禁的，则可前往幻梦官方群主动向幻梦开发组说明情况，幻梦开发组会在查证后删除封禁记录（不是单单的解封，是直接删除记录）且给予误封补偿。以下是参考补偿：
- 给予您基础补偿500额度和2000点好感度。
- 从您被封禁到解封的时间段内，每被误封禁1天（不足1天按1天算），额外给予您500额度和1000点好感度作补偿。这条补偿最多不超过2000额度和5000点好感度。
- 根据对您造成的实际影响，看情况额外补偿您非期限或期限道具。
- 尽快消除对您的其他影响并恢复声誉，如可能的舆论压力等。
单次总补偿量不会超过5000额度和12500点好感度（包括道具的价值）。

## 幻梦对我的拉黑会自动解除吗？
不会。除您主动申诉解封外，否则幻梦对你的拉黑永远不会解除。
请小人类尽快通过幻梦下发的消息指引去申请解封，真的做错了什么不重要，去群里如实说明情况并道歉后，幻梦开发组会视情况给予解封（99.9%的人去群里申诉都得到了解封），若你不来申诉那么永远也别想解哦。
