### 来自开发组的一段话

::: tip
幻梦始终兼顾着 `免费使用，为爱发电` 的原则，我们不对任何功能进行 收费 ，更不会让幻梦成为所谓的“商业产品”。服务器的机子目前由我们（幻梦开发组）自费购买，我们从开发幻梦开始到现在累计已经消耗了超过10,000块钱。随着幻梦的发展，开发成本越来越高，而幻梦开发者们的生活收入几乎是不变的，这使得极端情况下开发者们不得不拿出原先要吃饭的钱来投入运营。幻梦的发展不是一帆风顺的，每天幻梦开发者都能在后台记录里查到 恶意辱骂、恶意非法调用功能 的用户，这其中，又有多少个用户懂得开发者们的困难呢？即便这样，幻梦开发者们仍然坚持运营着幻梦，坚持把原先要付费的接口、或项目、或功能，免费地带给所有人。为幻梦开发者买一杯奶茶，可以让幻梦开发者有着更好的心情敲代码，可以让幻梦开发者开心一整天！并且你的建议或者吐槽会被更快地被幻梦开发者们回复哦。
:::

### 支持方案
幻梦坚持公益性质的运营方式，但正如大家所知：总要吃点饭。小人类为幻梦赞助的所有金额，均会全权用于支付运营幻梦所产生的成本费用，以及作者们的基本生活开支！

::: info
提示: 投喂与不投喂的用户在游玩幻梦的核心体验上是完全一致的。投喂幻梦并不会为小人类带来任何小游戏的属性加成或特殊特权。
:::
