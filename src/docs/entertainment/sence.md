# 好感度

## 好感度是什么？
幻梦好感度是衡量幻梦与你之间的亲密程度的重要指标。

好感度等级越高，代表幻梦越加亲近你，你在游玩幻梦时的局外加成会越高：
- 好感度越多，你的体力上限越高、体力恢复速度越快；
- 好感度越多，你的单次打卡能获得的额度和好感度越多，运势抽到的幸运值越高。
- 好感度越多，在游玩部分小游戏不幸失败时，失去的额度越少。

## 好感度如何提升？
游玩绝大部分的幻梦的小游戏均可以提升好感度，如速算、破碎、圈小猫等！
