# 恶臭数字论证器 
**{#9A38DF}经典玩法回归~{}**

## 主要玩法 
- 任何数字，都可以组成114514算式的形式出现！ 
- 输入任何数字，幻梦都会返回包含114514数字的算式。 
如: 
- **1** = 11/(45-1)*4 
- **3** = 11*-4+51-4 
- **456** = 1*1+451+4 
以此类推...

## 与老版本的代码翻新内容 
1. 支持小数输入 
   - 旧版的论证器不支持小数输入，只支持整数输入，如果用户输入了小数则系统会自动将数字向上取整。新版相较于旧版在代码上有了改进，使得在代码层面上支持了用户输入小数{#F98825}（由于浮点数精度问题，目前只支持至多小数点后两位）{}。 
          
![](/img/Homo/Homo2.jpg){width="200"}

2. 上限扩大 
   - 旧版的论证器在代码层面上最多只支持27位数字长度，超过则报错。新版则支持到至多54位长度。 
   {#2972F4}（这只是代码层面的支持，实际上幻梦开发组为了防止该功能过于刷屏，在群聊环境、频道环境内分别设置了字符输入上限，如群聊环境的字符输入上限默认15位长度，超过上限则自动截断后面的数字。若要体验更多的输入上限可前往单聊环境游玩）{} 
   {#45B076}（投喂用户输入字符上限长度额外+5）{} 
![](/img/Homo/Homo3.png){width="300"}

3. 新增多项字母支持 
   - 支持如 **π、e、φ** 等字母输入{#DD4097}（仅限单一字母，不支持组合输入）{}。
