# 多人模式（开发中）

## 对战模式（V.S.）

介绍: 由一名玩家扮演抓猫者，一名玩家扮演猫咪。在场地里面体验真人圈真猫\~
机制如下:

- 场地上存在两只小猫。
- 游戏开始后，由抓猫者先手放置一个墙壁，然后另一名玩家可以指定一只猫咪，令它受玩家方向指引移动一格，另一只小猫随机移动一格。移动后，场地的随机位置将生成一堵墙壁。
- 每次猫咪玩家操作一步，可获得积分2。
- 每次抓猫者玩家操作一步，可获得积分1。

## 合作模式（Co-op）

介绍: 两名玩家共同来到场地，一起合作抓两只猫。
玩法：

- 游戏开始时，场地会依据选择的难度来随机生成若干墙壁。
- 由玩家A开始（创建房间玩家）放置墙壁。
- 玩家A放置墙壁后，由玩家B（加入房间玩家）放置第二个墙壁。
- 玩家B放置了墙壁后，两只小猫同时移动一步。
- 接着由玩家A放置第三个墙壁，再由玩家B放置第四个墙壁，以此类推......

所有难度：

1. 🟩简单
   - 游戏开始时，将在随机位置生成12个墙壁。
   - 积分倍率：1.5x
   - 两只小猫具有基本的逃脱策略。

2. 🟨普通
   - 游戏开始时，将在随机位置生成11个墙壁。
   - 积分倍率：1.1x。
   - 小猫具有标准逃脱策略的AI，会使用标准算法计算逃离到边界的最快路线来逃离。
   - 当场地上有一只小猫被圈住或逃离到场地外时，剩余的一只猫可以一次性走两步。

3. 🟥困难
   - 游戏开始时，将在随机位置生成10个墙壁。
   - 积分倍率：0.8x。
   - 小猫具有更高级的逃脱策略。在行动时，小猫会主动分析小人类包围的意图，并在接下来的行动中，使用标准算法计算逃离到边界的最快路线来逃离。
   - 当当场地上有一只小猫被圈住或逃离到场地外时，剩余的一只猫可以一次性走两步。
