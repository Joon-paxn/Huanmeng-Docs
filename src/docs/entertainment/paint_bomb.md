# 💥 油漆炸弹

## 前期剧情

> 欢迎勇士来到神秘密室！
> 你能来到这里，肯定是事先做了不少准备的吧？那么，就请你脱掉脚上的鞋子和袜子，光着脚站在地面上的一个方块开始。

## 玩法规则

1. 自由选择从哪个格子开始，你也可以让系统自动选择（专门给选择困难症的）。
2. 每次移动你只能移动一格，不可以 ~~跳着走，斜着走，隔着走，往回走，走到地图外~~。
3. 移动过程你可能会踩到不同的东西，这些东西都会对你接下来的寻宝有影响：有可能是宝藏，也有可能让你直接结束寻宝...
4. 初始步数每位用户大大也是不同的哦，根据 [每日运势](/docs/entertainment/fortune)计算。

根据踩到不同东西，你将分别获得不同的效果，如下：

- **油漆**：将被底下冒出来的油漆紧紧粘住脚无法逃脱，不得不花费额度呼救。结束游戏并失去一定额度。
- **触手**：被突然伸出的触手包裹，挣扎耗费了许多精力，可行走步数 {red}-3{}。
- **宝藏**：发现了宝藏！获得额度奖励。
- **一次性凉鞋**：穿上后可抵消一次油漆和触手的伤害。已经持有且重复捡到不计算。
- **无皮橘子**：踩到溅了你一身，并使你有额外的橘子味！
  - 当接下来的寻宝踩到触手时，仅 {red}-2{} 步数。
  - 踩到宝藏，**双倍奖励**！
  - 踩到油漆，**双倍惩罚**。已经有该效果再次踩到的不计。
- **香皂**：抹上后精力更加旺盛了！
  - 可行走的步数 {green}+3{}。
  - 但当接下来的寻宝踩到触手时，触手也喜欢这个味道，因挣扎困难额外 {red}-2{} 步数。

## 隐藏文档

以下是新版油漆炸弹的草稿：

初始步数：14步，小人类可以花费额度和梦晶来增加初始步数，最多增加6步。
(50/200/400/1000/30/50)

初始步数会根据今日运势动态调整，如下：

- 欧皇，+1步
- 满级欧皇，+2步
- 超皇，+3步
- 非酋，-1步
- 超级非酋，-2步

新增理智值，每局开始时，将有100点理智值，理智值会受局内捡到的道具影响。
理智值不会低于0点，不会高于120点。

### 拾取宝藏：

- 将获得8额度。
- 有概率获得道具。道具池待开发者小布丁评估后开放给大家看。（注意，必须是自然走完步数才能将这些道具带走！如果因为油漆或者触手导致步数被动扣完的，是带不走这些道具的哦）
- 理智值大于80点时，每高于1点理智值，获得的额度额外增加1.5%，上限+50%。
- 理智值低于60点时，每低于1点理智值，获得的额度减少2%，上限-90%。

### 拾取一次性凉鞋：

- 可抵消一次油漆和触手的伤害。
- 重复捡到一次性凉鞋时，可累计抵消的次数，但最多不会超过2次。
- 若理智值小于30点，则无法拾取。

### 踩到香皂：

- 可行走步数+2。
- 如果理智值大于30点，可行走步数额外+1；大于100点，进一步+1。
- 恢复理智值5点。如果理智值低于90点，额外+5点理智值恢复；低于30点，进一步+5点。
- 在接下来的6步，踩到触手后，可行走步数额外减少2步，同时额外增加30%理智值扣除。

### 踩到无皮橘子：

- 在接下来的5步，拾取宝藏时，额外增加100%额度，并且根据获得额度的40%，在结算时额外获得好感度。
- 在接下来的5步，踩到油漆后，根据"踩到油漆 - 有橘子效果"来计算。
- 重复踩到无皮橘子时，可生效的步数叠加，但最多不会超过12步。
- 若理智值小于50点，则减少1步可行走步数；小于30点，额外减少1步；小于10点，额外减少1步。

### 踩到油漆 - 无橘子效果：

- 减少15步可行走步数。
- 减少30-40点理智值。
- 理智值扣除后，如果理智值仍高于30点，则每拥有10点理智值（从30点开始计算），恢复1点可行走步数。

### 踩到油漆 - 有橘子效果：

- 减少10步可行走步数。
- 减少40-50点理智值。
- 如果理智值扣除完后低于60点，则根据剩余的理智值多少，额外增加理智值扣除（上限50%）。
- 如果理智值扣除完后高于60点（包括50点），则根据剩余的理智值多少，额外减少理智值扣除（上限30%）。
- 理智值扣除后，如果理智值仍高于30点，则每拥有10点理智值（从30点开始计算），恢复1点可行走步数。

### 踩到触手：

- 可行走步数-3。
- 减少20-25点理智值。
- 如果一局内重复踩到触手，则：
  - 理智值大于60点，不变。
  - 理智值小于等于60点，额外增加50%的理智值扣除，但本次可行走步数只-2。
- 在接下来的5步，若踩到了油漆，额外减少2步可行走步数（如有）。

### 传送门：

- 消耗1点可行走步数进入或出来。
- 若小人类理智值低于40点，则在进入或出来传送门时有30%的概率额外减少1\~3点理智值。
- 进入传送门后，小人类将移动至场景的下层，根据下层的场地大小，为小人类分配初始可在下层中行走的步数（这个步数和上层的可行走步数是独立分开的）。在下层中或许有独特的其他事件等待小人类发现哦\~

### 本局结算时：

- 保底获得15点好感度。如果拾取到了额度，将额外基于获得额度的60%，来获得好感度的奖励。
- 每局获得的额度最多不会超过35，好感度不会超过40点。

### 规则新增：

- 理智值低于30点时，每行走一步，均有一定的概率额外消耗一步，理智值越低，这个概率就越大。范围是5%-100%。
- 寻宝失败时，如果理智值低于50点，则会额外扣除额度来进行"治疗"。理智值越低，额外扣除的额度数越多。范围是10%-800%。（到时候可前往新的加成页面，通过升级对应的加成来减少这个范围）
