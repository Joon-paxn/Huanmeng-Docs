# 接龙游戏
## 前言
接龙是一种常见的玩法，小人类需要按照一定的规则，如从某个词语或成语的最后一个字作为下一个词语或成语的开头，进行连续的接龙
这类益智游戏不仅可以锻炼词汇量，还能提高小人类的思维敏捷度和语言表达能力‌。

## 如何开始？
1. {blue}@幻梦{}并输入指令 "**/接龙游戏**" 列出主菜单。
> 应QQ规则，必须手动@幻梦，幻梦才能处理指令，不能直接打"@幻梦"这几个字！
 
![接龙主菜单](/img/word_chain/main.png){.hm-center-img width="300"} 

2. 选择你要在群内发起的接龙游戏，布丁{pink}（幻梦作者）{}就以单词为例~ 点击**开始**按钮即可开始接龙。

![接龙开始](/img/word_chain/start.png){.hm-center-img width="300"}  

3. 接龙开始后，幻梦会随机抽取一个单词末尾，从图内得知，第一个单词末尾是 **e**，所以小人类需要想一个 **e** 开头的单词来接龙，布丁就以单词 **event**{#8f8f8f}（翻译: 活动）{}来举例~

![接龙英语单词示例](/img/word_chain/type_to_chain.png){.hm-center-img width="300"}  

4. 如果单词符合规则，则接龙成功，反之接龙失败。

![接龙成功示例](/img/word_chain/chain_success.png){.hm-center-img width="300"}  

5. 时间到后，幻梦会结算每个人接龙的数据来发放奖励。

![接龙结算](/img/word_chain/settlement.png){.hm-center-img width="300"}
