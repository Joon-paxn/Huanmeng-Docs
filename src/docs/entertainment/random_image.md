[幻梦Bot交互功能声明](https://docs.qq.com/aio/p/sc7sp6juoyjk6t3?p=udetuMYDi4vTSL3LpBVJKZ\&nlc=1)

## 主要玩法

每天你都可以通过这个玩法来随机获取可能喜欢的二次元美图！<br>
也许你会喜欢上这些美图，从而将这些美图设为你的电脑或手机壁纸，或是做成头像。<br>
反正，总有一张适合你！每一页的版本幻梦开发团队都会从网络上下载合适的图片并更新到随机图的存储总量，让图片获取更加的随机和具有未知性！<br>

## 随机图的特点

1. **完全随机**，每一张图片都有新惊喜。<br>
2. **原图高品质**。绝大部分的图收录进幻梦都会不压缩的按照原图的规格发送出去，让小人类可以把这张图下载下来当作高清壁纸、收藏用。<br>

## 获取规则

由于图片集由幻梦开发团队和热心用户收集提供，请小人类理解我们收集图片时所包含的时间、人力成本，所以当一天获取随机图次数超过一定次数时，幻梦会对小人类获取图做作为获取图片费用\~<br>

| 获取次数      | 缴费额度 |
| :-------- | :--- |
| 第1\~20次   | 免费   |
| 第21\~40次  | 12   |
| 第41\~100次 | 10   |
| 第101次及以上  | 9    |

第1次到第20次获取免费，第21次及以上获取需要消耗额度。若小人类额度不足以消耗，将无法再获取图片。<br>
投喂用户每天可以额外获取2张图片，即第1次\~第22次获取是免费的，[点击这里了解](/docs/support)。<br>

小人类可以在随机图设置内启用 **好感度抵扣** 功能！启用这个功能时，每天第21次及以上获取随机图可以消耗一定量的好感度来抵扣\*\*至多50%\*\*的额度消耗。这个功能在一定程度上可以缓解小人类持有额度不多的问题。<br>
由于高等级的用户在推进好感度时，通常获得的好感度比低等级的用户多，所以具体消耗多少好感度根据下表标准以达到平衡：<br>

| 好感度等级 | 抵扣1额度时，所消耗的好感度比例 |
| :---- | :--------------- |
| D级    | 75%              |
| C级    | 150%             |
| B级    | 250%             |
| A级    | 400%             |

1. 如本次获取随机图需消耗{orange}5额度{}，用户的好感度等级为{orange}D1{}，那么将消耗 {orange}2.5 * 0.75 = 1.875{} 点好感度来抵扣50%的额度消耗，实际消耗{orange}2.5额度、1.875点好感度{}。<br>
2. 如本次获取随机图需消耗{orange}6额度{}，用户的好感度等级为{cyan}C1{}，那么将消耗 {cyan}3 * 1.5 = 4.5{} 点好感度来抵扣50%的额度消耗，实际消耗{cyan}3额度、4.5点好感度{}。<br>
3. 如本次获取随机图需消耗{orange}8额度{}，用户的好感度等级为{blue}B2{}，那么将消耗 {blue}4 * 2.5 = 10{} 点好感度来抵扣50%的额度消耗，实际消耗{blue}4额度、10点好感度{}。<br>
4. 如本次获取随机图需消耗{orange}8额度{}，用户的好感度等级为{red}A3{}，那么将消耗 {red}4 * 4 = 16{} 点好感度来抵扣50%的额度消耗，实际消耗{red}4额度、16点好感度{}。<br>

- 当用户好感度低于0时，将无法完成抵扣，额度正常消耗。<br>
- **仅背包等级大于5**的用户的小人类可以开启好感度抵扣功能。<br>

## 随机图设置

### 优化模式是什么？

当小人类获取抽取的随机图片时，图片有可能会因为QQ自身拦截限制而无法发送成功。启用优化模式后，将利用内置优化路径来辅助消息能够发送成功，让小人类获取图片更顺畅\~<br>

### 图片画风可以改吗？

- 暂不支持修改图片画风。目前幻梦开发团队收录图片以二次元为主，不会包含三次元元素。<br>
- 后续会考虑开发“自定义二次元风”风，比如小人类喜欢看女孩子的照片，那么幻梦更新后会随机获取到标签包含女孩子的图片等\~<br>

### 图片适龄段是什么？调了有什么用？

目前仅**背包等级大于18**的小人类可以调整图片适龄阶段。<br>
图片适龄阶段代表该图片适合哪些年龄阶段的小人类观看和欣赏。<br>
默认图片适龄阶段为 🟦 大众级（9+，代表该张图片推荐9岁以上的小人类观看欣赏）。<br>
合理的调整图片适龄阶段可以让小人类获取的图片更贴合你当前年龄阶段的喜好。<br>
你可以将适龄阶段调整为 🟩 家长指导级（12+）。这代表幻梦将有可能给你显示更多裸露、暴力、血腥或黑暗风格的图片。
