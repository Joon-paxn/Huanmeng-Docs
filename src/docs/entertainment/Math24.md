# 二十四点

## 主要玩法
复刻小学时期的二十四点卡牌玩法！<br>
玩法很简单，游戏开始后，幻梦将会给出 **4** 个数字，小人类呢，需要算出这 **4** 个数字是否可以组成 **24** 点，即通过加减乘除等四则运算得出最终结果是否可以等于数字 **24** 。
如果回答正确，则获得额度和好感度奖励，但如果回答错误，将根据回答时长扣除额度作为惩罚。
::: warning
**不要蒙题哦，后果自负！**
:::
## 如何游玩？

1. 点击第一个按钮开始游戏。<br>
![#*px *px](/img/Math24-1.png)
2. 此时查看幻梦给的4个数字。小人类需要通过各种方式（比如脑海中口算，在草稿纸上打草稿，与身边的小伙伴或群友讨论等）来推理这4个数字是否可以通过四则运算来组成数字24。
![#*px *px](/img/Math24-2.png)   
3. 如果所给的4个数字可以组成24点算式，但是小人类回答了【不能】，则判定回答错误。同理，如果给的4个数字不能组成24点算式，但小人类回答【可以】，也会判定回答错误。耐心计算，想象小人类最后可以成功的！
![#*px *px](/img/Math24-3-1.png) ![#*px *px](/img/Math24-3-2.png)
4. 小人类可以点击主界面或回答结果界面的【24点数据】按钮来进入数据与设置界面，在这里，小人类可以看到自己的24点游玩数据，也可以点击下方的按钮调整一些设置，如【难度】和【辅助设置】！难度在下面的文章里有讲~
![#*px *px](/img/Math24-4.png)
## 难度，奖励和惩罚
小人类可以随时在二十四点玩法里选择难度，难度越高，奖励越好，但惩罚也越狠。<br>
默认难度为**普通**难度，你也可以选择更高的难度挑战自己。<br>
**🟩简单难度**：更容易抽到个位数数字，算式更简单。胜利获得的额度{red}-30% {}，失败失去的额度{green}-80%{}。推荐新手和不熟悉二十四点玩法的小人类选择。<br>
**🟨普通难度**：标准难度。<br>
**🟥困难难度**：算式生成更为谨慎。胜利获得的额度{green}+30% {},但是失败失去的额度{red}+100%{}。推荐熟悉二十四点玩法的小人类选择。   
**🔥极限难度**：算式生成机制与困难难度相同，但是算式需要自己打入。胜利获得的额度{green}+50% {},失败失去的额度{red}+100%{}.推荐精通二十四点玩法的小人类选择哦。<br>
> 以普通难度为基准，每局胜利可至多获得 16 额度，每局失败失去至多 8 额度（不包含蒙题惩罚，其他加成等）。
