# 幻梦的达尔文游戏

作为娱乐分区最流行的机器人之一，幻梦提供了大量娱乐小游戏，它们有一个官方的名字：**达尔文游戏**。

## 共享机制

每个小人类除了背包之外，还有额度和好感度。游玩达尔文游戏的过程中，胜利会增加额度和好感度，而失败会扣除额度和好感度哦。

## 体力

为了防止沉迷，游玩达尔文游戏都需要付出一定的体力，当体力不足时就不能游玩哦~

每种达尔文游戏都会消耗不同的体力，给予/扣除的额度和好感度也不同，这些数据可以参考各项游戏的页面。
